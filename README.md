# CORAL

*COastal Response emulator & Adaptation Learning* | *Coastal Options for Resilience & Adaptation Learning*

A machine learning framework for rapid evaluation of coastal flood response to bathymetric interventions under sea level rise.

## Project Overview

CORAL develops ML surrogates for physics-based hydrodynamic models to:
- Correct systematic biases in storm surge predictions
- Emulate compound flooding response to infrastructure interventions
- Enable rapid exploration of coastal adaptation pathways
